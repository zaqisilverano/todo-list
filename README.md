## TODO LIST APP

Install main dependencies:

```bash
npm install
# or
yarn install
```

Install native dependencies:
```bash
npx pod-install
```

Run on Android:

```bash
npm run android
# or
yarn android
```

Run on IOS:

```bash
npm run ios
# or
yarn ios
```

Run the unit test:

```bash
npm run test
# or
yarn test
```

## Tech Stack

- Bare React Native
- Typescript
- Expo Local-Authentication
